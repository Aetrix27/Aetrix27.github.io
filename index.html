In Stock Gaming Hardware Checker

Checks if gaming items are in stock from Amazon, using a web scraper API called Requests HTML. It uses efficient for loops to run the program as fast as possible. In recent months, it has been hard to find graphics cards and gaming consoles due to bots and scalpers taking them as soon as they are available, and the global silicon chip shortage. This gives power back to the people.

Download: Github Repository

EasyHealth (Website and Separate iPhone App)

Easily Share Medical Info with your personal doctor by uploading documents and sending information. Patients can sign up and login as well as Doctors. The website can tell whether a patient or user is authenticated based on the sign up process. If a patient sends a document to their doctor, their ID is automatically tied to their document. The SQL relational database then ties the patient and the doctor. Doctors can also send documents and the same happens to them.

Technologies: Created with Python, HTML, Jinja2, Flask, SQL and Bootstrap
Live link: EasyHealth
The separate iPhone App uses Swift to allow users to be created, and send messages. Messages are currently able to be sent and users can be created and authenticated, and it is a work in progress.

Technologies: Swift, Firebase, Cocoapods

Get Big Videogame

In this game, the player plays against a computer to collect dots to get bigger before the computer does. If the player is bigger than the computer, they can eat them, and then win the game. The computer uses vectors to follow the player to their real-time position. They can select to play again or quit in the menu screens.

Technologies: Python, Pygame
